### 62. 文件上传（通用）

**接口：** `POST /api/upload`

**权限：** 需认证

**请求体（multipart/form-data）：**
```
file: <binary>
type: avatar|pet_media|checkin_media|application_document
```

**参数说明：**
| 参数 | 类型 | 必填 | 说明 |
|-----|------|-----|------|
| file | file | Y | 待上传文件 |
| type | enum | Y | 文件类型，用于验证和路径分类 |

**文件限制：**
| 类型 | 允许格式 | 最大大小 |
|------|---------|---------|
| avatar | jpg, png, webp | 5MB |
| pet_media | jpg, png, webp, mp4, webm | image: 10MB, video: 100MB |
| checkin_media | jpg, png, webp, mp4, webm | image: 10MB, video: 100MB |
| application_document | pdf, jpg, png | 10MB |

**成功响应：**
```json
{
  "code": 0,
  "message": "上传成功",
  "data": {
    "url": "https://example.com/uploads/avatar/user-1001.jpg",
    "filename": "user-1001.jpg",
    "size": 123456,
    "mime": "image/jpeg"
  }
}
```

**业务规则：**
- 调用对象存储服务（COS/OSS/MinIO）
- 返回 CDN 外网可访问地址
- 可选：文件病毒扫描
- 可选：图片内容审核（敏感内容识别）

### 63. 获取图片验证码

**接口：** `GET /api/public/captcha`

**权限：** 无需认证

**查询参数：**
```
?timestamp=1705750800000
```

**成功响应：**
```json
{
  "code": 0,
  "message": "success",
  "data": {
    "captchaId": "abc123def456",
    "image": "data:image/png;base64,iVBORw0KGgoAAAANSUhEUgAAAA...",
    "expiresIn": 300
  }
}
```

**业务规则：**
- 生成随机图片验证码
- captchaId 和答案存储在 Redis，有效期 5 分钟
- 返回 base64 编码的图片或图片 URL

### 64. 发送验证码（邮件/短信）

**接口：** `POST /api/public/send-code`

**权限：** 无需认证

**请求体：**
```json
{
  "phone": "13800000001",
  "type": "register",
  "captchaId": "abc123def456",
  "captchaAnswer": "1234"
}
```

或

```json
{
  "email": "user@example.com",
  "type": "reset_password",
  "captchaId": "abc123def456",
  "captchaAnswer": "1234"
}
```

**参数说明：**
| 参数 | 类型 | 必填 | 说明 |
|-----|------|-----|------|
| phone | string | N | 手机号（若phone和email都传，优先phone） |
| email | string | N | 邮箱 |
| type | enum | Y | 用途：register、reset_password、bind_phone、verify_email |
| captchaId | string | Y | 验证码ID |
| captchaAnswer | string | Y | 用户输入的验证码答案 |

**成功响应：**
```json
{
  "code": 0,
  "message": "验证码已发送，有效期10分钟"，
  "data": {
    "expiresIn": 600,
    "code": "1234"
  }
}
```

**业务规则：**
- 先验证图片验证码
- 同一手机/邮箱 1 分钟内最多发送 1 次
- 同一手机/邮箱 1 小时内最多发送 5 次
- 验证码有效期 10 分钟
- 调用短信/邮件服务商 API（阿里云、腾讯云等）

### 65. 验证码验证

**接口：** `POST /api/public/verify-code`

**权限：** 无需认证

**请求体：**
```json
{
  "phone": "13800000001",
  "code": "123456"
}
```

**成功响应：**
```json
{
  "code": 0,
  "message": "验证成功",
  "data": {
    "token": "temp_token_abc123...",
    "expiresIn": 1800
  }
}
```

**业务规则：**
- 验证码正确后返回临时 token，用于后续重置密码等操作
- 临时 token 有效期 30 分钟，仅可用于特定接口
