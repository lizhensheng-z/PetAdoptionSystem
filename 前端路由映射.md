## 附录：前端路由映射

### 用户端路由示例

```
/                          → 首页（推荐流）
/pets                      → 宠物列表
/pets/:id                  → 宠物详情
/apply/:petId              → 申请表单
/applications              → 我的申请
/applications/:id          → 申请详情
/checkins                  → 我的打卡
/checkins/create           → 发布打卡
/favorites                 → 我的收藏
/profile                   → 个人资料
/preference                → 偏好设置
/credit                    → 信用中心
/login                     → 登录
/register                  → 注册
```

### 机构端路由示例

```
/org                       → 机构首页
/org/pets                  → 宠物管理列表
/org/pets/create           → 创建宠物
/org/pets/:id/edit         → 编辑宠物
/org/applications          → 申请管理
/org/applications/:id      → 申请详情
/org/adoptions             → 领养记录
/org/followup              → 回访管理
/org/profile               → 机构资料
/org/statistics            → 统计数据
```

### 管理端路由示例

```
/admin                     → 管理首页
/admin/dashboard           → 数据看板
/admin/pet-audits          → 宠物审核
/admin/pet-audits/:id      → 审核详情
/admin/users               → 用户管理
/admin/users/:id           → 用户详情
/admin/tags                → 标签管理
/admin/config              → 系统配置
/admin/notices             → 公告管理
/admin/audit-logs          → 审计日志
```
