## 推荐与搜索接口

### 23. 获取个性化推荐宠物

**接口：** `GET /api/recommend`

**权限：** 无需认证（有登录用户优先使用用户偏好，无登录返回热门）

**查询参数：**
```
?pageNo=1&pageSize=10&lng=116.4&lat=39.9&includeDistance=true
```

| 参数 | 类型 | 必填 | 说明 |
|-----|------|-----|------|
| pageNo | int | N | 页码，默认1 |
| pageSize | int | N | 每页条数，默认10 |
| lng | decimal | N | 用户经度（用于附近权重） |
| lat | decimal | N | 用户纬度 |
| includeDistance | boolean | N | 是否返回距离，默认true |

**成功响应：**
```json
{
  "code": 0,
  "message": "success",
  "data": {
    "list": [
      {
        "id": 2001,
        "name": "小花",
        "species": "CAT",
        "breed": "英短",
        "gender": "FEMALE",
        "size": "M",
        "coverUrl": "https://...",
        "orgName": "爱心救助站",
        "tags": [
          { "id": 101, "name": "亲人", "type": "PERSONALITY" },
          { "id": 102, "name": "安静", "type": "PERSONALITY" }
        ],
        "matchScore": 92,
        "matchReason": "符合您的偏好：物种匹配、体型匹配、性格标签命中",
        "distance": 2.5,
        "location": {
          "lng": 116.4,
          "lat": 39.9,
          "address": "北京市朝阳区xxx"
        },
        "isFavorited": false
      }
    ],
    "pageNo": 1,
    "pageSize": 10,
    "total": 50,
    "totalPages": 5,
    "recommendReason": "根据您的偏好和相似用户的选择推荐"
  }
}
```

**业务规则：**
- 若已登录，从 JWT 获取 user_id，查询 sys_user.preference_json 和用户行为数据
- 计算推荐分：`final_score = α * match_score + β * cf_score + γ * distance_score + δ * freshness`
- 返回 TOP 10 * pageSize 条（分页）
- 若无登录用户，返回最新发布或热门宠物

### 24. 附近宠物列表（地理位置优先）

**接口：** `GET /api/recommend/nearby`

**权限：** 无需认证

**查询参数：**
```
?lng=116.4&lat=39.9&distance=50&pageNo=1&pageSize=10&species=CAT&keyword=
```

| 参数 | 类型 | 必填 | 说明 |
|-----|------|-----|------|
| lng | decimal | Y | 用户经度 |
| lat | decimal | Y | 用户纬度 |
| distance | int | N | 搜索半径（km），默认50 |
| pageNo | int | N | 页码 |
| pageSize | int | N | 每页条数 |
| species | enum | N | 物种过滤 |
| keyword | string | N | 关键词过滤 |

**成功响应：**
```json
{
  "code": 0,
  "message": "success",
  "data": {
    "list": [
      {
        "id": 2001,
        "name": "小花",
        "species": "CAT",
        "breed": "英短",
        "coverUrl": "https://...",
        "orgName": "爱心救助站",
        "distance": 2.5,
        "location": {
          "lng": 116.4,
          "lat": 39.9,
          "address": "北京市朝阳区xxx"
        },
        "tags": [
          { "id": 101, "name": "亲人", "type": "PERSONALITY" }
        ],
        "isFavorited": false
      }
    ],
    "pageNo": 1,
    "pageSize": 10,
    "total": 85,
    "totalPages": 9,
    "searchCenter": {
      "lng": 116.4,
      "lat": 39.9
    },
    "searchRadius": 50
  }
}
```

**业务规则：**
- 使用 Haversine 公式或 MySQL 空间索引计算距离
- 可缓存同城/同坐标的附近宠物列表到 Redis
- 返回结果按距离升序排列

### 25. 宠物搜索（高级筛选）

**接口：** `GET /api/pets/search`

**权限：** 无需认证

**查询参数：**
```
?keyword=小花&species=CAT&breed=英短&gender=FEMALE&sizeMin=1&sizeMax=3&ageMin=12&ageMax=120&sterilized=true&vaccinated=true&tags=101,102&city=北京&lng=116.4&lat=39.9&distance=50&pageNo=1&pageSize=10&sortBy=distance&order=asc
```

（参数同 [获取宠物列表](#7-获取宠物列表游客用户通用)）
