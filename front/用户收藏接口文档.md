# 用户收藏接口 - 前端对接文档

> 本文档提供用户收藏管理相关的所有后端接口，包括添加收藏、取消收藏、获取收藏列表、收藏状态检查等，方便前端开发对接。

---

## 1. 添加收藏

**接口说明**：用户收藏指定的宠物

**请求方式**：`POST`

**请求路径**：`/api/favorites/{petId}`

**请求头**：
```
Authorization: Bearer <access_token>
Content-Type: application/json
```

**路径参数**：

| 参数 | 类型 | 必填 | 说明 |
|------|------|------|------|
| petId | Long | 是 | 宠物ID |

**请求示例**：
```
POST /api/favorites/123
```

**响应示例**：
```json
{
  "code": 10000,
  "message": "收藏成功",
  "data": null
}
```

## 4. 收藏切换（收藏/取消收藏）

**接口说明**：切换宠物的收藏状态，已收藏则取消，未收藏则添加

**请求方式**：`POST`

**请求路径**：`/api/favorites/toggle`

**请求头**：
```
Authorization: Bearer <access_token>
Content-Type: application/json
```

**请求体**：
```json
{
  "petId": 123
}
```

**请求字段说明**：

| 字段 | 类型 | 必填 | 说明 |
|------|------|------|------|
| petId | Long | 是 | 宠物ID |

**响应示例**：
```json
{
  "code": 10000,
  "message": "操作成功",
  "data": {
    "favorited": true,
    "message": "收藏成功"
  }
}
```

**响应字段说明**：

| 字段 | 类型 | 说明 |
|------|------|------|
| favorited | Boolean | 当前收藏状态（true:已收藏，false:已取消） |
| message | String | 操作结果描述 |

## 6. 前端对接示例

### 6.1 添加收藏

```javascript
// 使用axios请求
import axios from 'axios';

async function addFavorite(petId) {
  try {
    const response = await axios.post(`/api/favorites/${petId}`, {}, {
      headers: {
        'Authorization': `Bearer ${localStorage.getItem('token')}`
      }
    });
    
    if (response.data.code === 10000) {
      console.log('收藏成功');
      return true;
    } else {
      console.error('收藏失败:', response.data.message);
      return false;
    }
  } catch (error) {
    console.error('请求失败:', error);
    return false;
  }
}

// 调用示例
addFavorite(123);
```

### 6.6 综合使用示例 - 收藏管理组件

```javascript
// 收藏管理Hook
import { ref, computed } from 'vue';
import { useAuthStore } from '@/stores/auth';

export function useFavorite(petId) {
  const authStore = useAuthStore();
  const isFavorited = ref(false);
  const loading = ref(false);

  // 检查收藏状态
  const checkStatus = async () => {
    if (!authStore.isLoggedIn) return;
    
    try {
      const response = await checkFavoriteStatus(petId);
      isFavorited.value = response;
    } catch (error) {
      console.error('检查收藏状态失败:', error);
    }
  };

  // 切换收藏
  const toggle = async () => {
    if (!authStore.isLoggedIn) {
      // 跳转到登录页面
      return { success: false, needLogin: true };
    }

    loading.value = true;
    try {
      const response = await toggleFavorite(petId);
      isFavorited.value = response;
      return { success: true, favorited: response };
    } catch (error) {
      console.error('收藏操作失败:', error);
      return { success: false, error: error.message };
    } finally {
      loading.value = false;
    }
  };

  return {
    isFavorited: computed(() => isFavorited.value),
    loading: computed(() => loading.value),
    checkStatus,
    toggle
  };
}

// 在组件中使用
import { useFavorite } from '@/composables/useFavorite';

export default {
  setup() {
    const { isFavorited, loading, toggle } = useFavorite(123);
    
    const handleFavorite = async () => {
      const result = await toggle();
      if (result.needLogin) {
        // 处理需要登录的情况
        this.$router.push('/login');
      }
    };

    return {
      isFavorited,
      loading,
      handleFavorite
    };
  }
};
```

        </div>

      :page-size="pageSize"
      :total="total"

    />
  </div>

<script setup>
import { ref, onMounted } from 'vue';
import { useRouter } from 'vue-router';

const router = useRouter();
const favorites = ref([]);
const currentPage = ref(1);
const pageSize = ref(10);
const total = ref(0);

const loadFavorites = async () => {
  try {
    const data = await getMyFavorites(currentPage.value, pageSize.value);
    favorites.value = data.list;
    total.value = data.total;
  } catch (error) {
    console.error('加载收藏列表失败:', error);
  }
};

const viewDetail = (petId) => {
  router.push(`/pets/${petId}`);
};

const removeFavorite = async (petId) => {
  try {
    await removeFavorite(petId);
    ElMessage.success('已取消收藏');
    loadFavorites();
  } catch (error) {
    console.error('取消收藏失败:', error);
  }
};

onMounted(() => {
  loadFavorites();
});
</script>

## 8. 错误码说明

| 错误码 | 说明 | 处理建议 |
|--------|------|----------|
| 10000 | 操作成功 | - |
| 10001 | 操作失败 | 检查请求参数或联系客服 |
| 20001 | 参数错误 | 检查请求参数格式 |
| 20004 | 参数校验失败 | 检查必填字段和字段格式 |
| 30001 | 未登录 | 请先登录 |
| 30002 | token已过期 | 请重新登录 |
| 30004 | 无权限访问 | 检查用户权限 |
| 40001 | 资源不存在 | 检查宠物ID是否正确 |
| 40002 | 重复收藏 | 该宠物已在收藏列表中 |
| 40003 | 收藏不存在 | 该收藏记录不存在 |
| 50000 | 系统异常 | 稍后重试或联系客服 |

## 9. 注意事项

1. **认证要求**：所有接口都需要在请求头中携带 `Authorization: Bearer <token>`
2. **重复收藏限制**：同一用户同一宠物只能收藏一次
3. **分页参数**：分页接口的 `pageNo` 从 1 开始
4. **时间格式**：所有时间字段格式为 `yyyy-MM-dd HH:mm:ss`
5. **数据脱敏**：手机号等敏感信息已脱敏处理
6. **并发处理**：建议添加加载状态和错误处理
7. **缓存策略**：收藏列表数据建议缓存3-5分钟
8. **实时更新**：操作后建议重新拉取数据确保一致性
9. **权限控制**：未登录用户无法使用收藏功能
10. **性能优化**：收藏列表建议实现虚拟滚动
