## 1. 页面功能概述

### 1.1 页面定位
用户管理页面是管理端的核心功能，用于系统管理员查看、搜索、筛选、管理所有平台用户（领养人/机构/管理员）。提供用户列表展示、详情查看、状态管理、权限分配等完整功能。

### 1.2 核心功能模块

| 模块 | 功能描述 | 权限要求 |
|------|---------|---------|
| **用户列表** | 展示所有用户信息，支持分页、排序 | `user:read` |
| **搜索筛选** | 按用户名、手机号、邮箱、角色、状态筛选 | `user:read` |
| **用户详情** | 查看用户完整信息，包括登录历史 | `user:read` |
| **状态管理** | 启用/禁用用户账号 | `user:update` |
| **角色管理** | 修改用户角色（USER/ORG/ADMIN） | `user:update` |
| **批量操作** | 批量启用/禁用/删除用户 | `user:update` |
| **导出功能** | 导出用户数据到Excel | `user:export` |

## 2. 数据库模型分析

### 2.1 用户表结构分析

基于提供的 `sys_user` 表结构，用户管理页面需要展示和管理以下字段：

| 字段名 | 类型 | 前端展示 | 可编辑 | 说明 |
|--------|------|----------|--------|------|
| `id` | BIGINT | ✅ | ❌ | 用户ID，主键 |
| `username` | VARCHAR(64) | ✅ | ❌ | 用户名，唯一 |
| `phone` | VARCHAR(32) | ✅ | ✅ | 手机号 |
| `email` | VARCHAR(128) | ✅ | ✅ | 邮箱地址 |
| `role` | VARCHAR(16) | ✅ | ✅ | 角色：USER/ORG/ADMIN |
| `avatar` | VARCHAR(512) | ✅ | ✅ | 头像URL |
| `status` | VARCHAR(16) | ✅ | ✅ | 状态：NORMAL/BANNED |
| `last_login_time` | DATETIME | ✅ | ❌ | 最后登录时间 |
| `create_time` | DATETIME | ✅ | ❌ | 创建时间 |
| `update_time` | DATETIME | ✅ | ❌ | 更新时间 |

### 2.2 角色权限映射

| 角色值 | 显示名称 | 描述 | 颜色标识 |
|--------|----------|------|----------|
| `USER` | 普通用户 | 宠物领养人 | 蓝色 |
| `ORG` | 机构用户 | 宠物救助机构 | 绿色 |
| `ADMIN` | 管理员 | 系统管理员 | 红色 |

### 2.3 状态映射

| 状态值 | 显示名称 | 颜色标识 | 操作 |
|--------|----------|----------|------|
| `NORMAL` | 正常 | 绿色 | 可禁用 |
| `BANNED` | 已禁用 | 红色 | 可启用 |

#### 4.1.3 核心方法

```typescript
// 加载用户列表
const loadUserList = async () => {
  try {
    tableData.value.loading = true
    const params = {
      page: pagination.value.page,
      pageSize: pagination.value.pageSize,
      ...searchForm.value
    }
    
    const response = await adminAPI.getUserList(params)
    tableData.value.list = response.data.list
    tableData.value.total = response.data.total
  } catch (error) {
    ElMessage.error('加载用户列表失败')
    console.error(error)
  } finally {
    tableData.value.loading = false
  }
}

// 搜索用户
const handleSearch = () => {
  pagination.value.page = 1
  loadUserList()
}

// 重置搜索
const resetSearch = () => {
  searchForm.value = {
    keyword: '',
    role: '',
    status: '',
    dateRange: []
  }
  handleSearch()
}

// 分页变化
const handlePageChange = (page) => {
  pagination.value.page = page
  loadUserList()
}

// 分页大小变化
const handleSizeChange = (size) => {
  pagination.value.pageSize = size
  pagination.value.page = 1
  loadUserList()
}

// 表格选择变化
const handleSelectionChange = (selection) => {
  selectedUsers.value = selection.map(item => item.id)
}

// 查看用户详情
const handleViewDetail = async (user) => {
  try {
    detailDialog.value.visible = true
    detailDialog.value.loading = true
    const response = await adminAPI.getUserDetail(user.id)
    detailDialog.value.data = response.data
  } catch (error) {
    ElMessage.error('获取用户详情失败')
  } finally {
    detailDialog.value.loading = false
  }
}

// 编辑用户
const handleEdit = (user) => {
  editDialog.value.form = {
    id: user.id,
    phone: user.phone,
    email: user.email,
    role: user.role,
    avatar: user.avatar,
    status: user.status
  }
  editDialog.value.visible = true
}

// 保存编辑
const saveEdit = async () => {
  try {
    editDialog.value.loading = true
    await adminAPI.updateUser(editDialog.value.form.id, editDialog.value.form)
    ElMessage.success('更新成功')
    editDialog.value.visible = false
    loadUserList()
  } catch (error) {
    ElMessage.error('更新失败')
  } finally {
    editDialog.value.loading = false
  }
}

// 切换用户状态
const toggleUserStatus = async (user) => {
  try {
    const newStatus = user.status === 'NORMAL' ? 'BANNED' : 'NORMAL'
    await adminAPI.updateUserStatus(user.id, newStatus)
    ElMessage.success('操作成功')
    loadUserList()
  } catch (error) {
    ElMessage.error('操作失败')
  }
}

// 批量启用用户
const batchEnableUsers = async () => {
  try {
    batchLoading.value = true
    await adminAPI.batchUpdateUserStatus(selectedUsers.value, 'NORMAL')
    ElMessage.success(`已启用 ${selectedUsers.value.length} 个用户`)
    selectedUsers.value = []
    loadUserList()
  } catch (error) {
    ElMessage.error('批量启用失败')
  } finally {
    batchLoading.value = false
  }
}

// 批量禁用用户
const batchDisableUsers = async () => {
  try {
    batchLoading.value = true
    await adminAPI.batchUpdateUserStatus(selectedUsers.value, 'BANNED')
    ElMessage.success(`已禁用 ${selectedUsers.value.length} 个用户`)
    selectedUsers.value = []
    loadUserList()
  } catch (error) {
    ElMessage.error('批量禁用失败')
  } finally {
    batchLoading.value = false
  }
}

// 批量删除用户
const batchDeleteUsers = async () => {
  try {
    await ElMessageBox.confirm(
      `确定要删除选中的 ${selectedUsers.value.length} 个用户吗？此操作不可恢复！`,
      '警告',
      { type: 'warning' }
    )
    
    batchLoading.value = true
    await adminAPI.batchDeleteUsers(selectedUsers.value)
    ElMessage.success(`已删除 ${selectedUsers.value.length} 个用户`)
    selectedUsers.value = []
    loadUserList()
  } catch (error) {
    if (error !== 'cancel') {
      ElMessage.error('批量删除失败')
    }
  } finally {
    batchLoading.value = false
  }
}

// 导出用户数据
const exportUsers = async () => {
  try {
    const response = await adminAPI.exportUsers(searchForm.value)
    const blob = new Blob([response.data], { type: 'application/vnd.ms-excel' })
    const url = window.URL.createObjectURL(blob)
    const link = document.createElement('a')
    link.href = url
    link.download = `用户数据_${new Date().toLocaleDateString()}.xlsx`
    link.click()
    window.URL.revokeObjectURL(url)
    ElMessage.success('导出成功')
  } catch (error) {
    ElMessage.error('导出失败')
  }
}

// 上传头像成功
const handleAvatarSuccess = (response) => {
  editDialog.value.form.avatar = response.data.url
}

// 初始化
onMounted(() => {
  loadUserList()
})
```

```typescript
  data: null,
  loginHistory: []

const router = useRouter()
```

```javascript
export function getUserList(params) {
  return http.get('/admin/users', params)
}

/**
 * 获取用户详情
 * @param {number} id - 用户ID
 * @returns {Promise} 用户详情数据
 */
export function getUserDetail(id) {
  return http.get(`/admin/users/${id}`)
}

/**
 * 更新用户信息
 * @param {number} id - 用户ID
 * @param {object} data - 更新数据
 * @returns {Promise} 更新结果
 */
export function updateUser(id, data) {
  return http.put(`/admin/users/${id}`, data)
}

/**
 * 批量更新用户状态
 * @param {number[]} userIds - 用户ID数组
 * @param {string} status - 新状态
 * @returns {Promise} 更新结果
 */
export function batchUpdateUserStatus(userIds, status) {
  return http.put('/admin/users/batch-status', { userIds, status })
}

/**
 * 批量删除用户
 * @param {number[]} userIds - 用户ID数组
 * @returns {Promise} 删除结果
 */
export function batchDeleteUsers(userIds) {
  return http.delete('/admin/users/batch', { userIds })
}

/**
 * 导出用户数据
 * @param {object} params - 查询参数
 * @returns {Promise} 文件流
 */
export function exportUsers(params) {
  return http.get('/admin/users/export', params, {
    responseType: 'blob'
  })
}

/**
 * 获取用户登录历史
 * @param {number} id - 用户ID
 * @returns {Promise} 登录历史数据
 */
export function getUserLoginHistory(id) {
  return http.get(`/admin/users/${id}/login-history`)
}
```

### 8.2 技术风险

| 风险项 | 风险等级 | 解决方案 |
|--------|---------|---------|
| 大数据量性能 | 中 | 使用分页 + 虚拟滚动 |
| 批量操作性能 | 中 | 限制单次操作数量 |
| 权限控制 | 低 | 前后端双重验证 |
| 数据安全 | 低 | 敏感信息脱敏显示 |
