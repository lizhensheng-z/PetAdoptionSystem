# 机构管理首页接口 - 前端对接文档

> 本文档提供机构管理首页（Dashboard）所需的所有后端接口，包括统计数据获取、待办事项、快捷操作等，方便前端开发对接。

---

## 1. 获取机构统计数据

**接口说明**：获取当前机构的关键统计数据，包括在养宠物数、待处理申请数、本月领养数、待回访数等

**请求方式**：`GET`

**请求路径**：`/api/org/dashboard/statistics`

**请求头**：
```
Authorization: Bearer <access_token>
Content-Type: application/json
```

**请求参数**：无

**响应示例**：
```json
{
  "code": 10000,
  "message": "操作成功",
  "data": {
    "totalPets": 25,
    "pendingApplications": 8,
    "monthlyAdoptions": 12,
    "pendingFollowups": 3,
    "totalAdoptions": 156,
    "publishedPets": 18,
    "draftPets": 7,
    "underReviewPets": 2
  },
  "timestamp": "2024-02-15 15:30:00"
}
```

**响应字段说明**：

| 字段 | 类型 | 说明 | 统计规则 |
|------|------|------|----------|
| totalPets | Integer | 在养宠物总数 | 状态为 PUBLISHED、APPLYING 的宠物 |
| pendingApplications | Integer | 待处理申请数 | 状态为 SUBMITTED、UNDER_REVIEW 的申请 |
| monthlyAdoptions | Integer | 本月领养数 | 当月 APPROVED 的申请数 |
| pendingFollowups | Integer | 待回访数 | 已领养且回访超期的宠物数 |
| totalAdoptions | Integer | 累计领养数 | 历史 APPROVED 的申请总数 |
| publishedPets | Integer | 已发布宠物数 | 状态为 PUBLISHED 的宠物数 |
| draftPets | Integer | 草稿宠物数 | 状态为 DRAFT 的宠物数 |
| underReviewPets | Integer | 审核中宠物数 | 状态为 PENDING_AUDIT 的宠物数 |

## 2. 获取待办事项列表

**接口说明**：获取当前机构的待办事项，包括待审核申请、超期回访提醒等

**请求方式**：`GET`

**请求路径**：`/api/org/dashboard/todos`

**请求头**：
```
Authorization: Bearer <access_token>
Content-Type: application/json
```

**请求参数**：

| 参数 | 类型 | 必填 | 说明 | 默认值 |
|------|------|------|------|--------|
| type | String | 否 | 待办类型筛选 | - |
| limit | Integer | 否 | 返回数量限制 | 10 |

**待办类型说明**：
- `application` - 待审核的领养申请
- `followup` - 待回访的宠物
- `audit` - 待审核的宠物发布

## 8. 前端对接示例

### 8.1 使用综合接口获取首页数据

```javascript
// 获取首页综合数据
async function loadDashboardData() {
  try {
    const response = await axios.get('/api/org/dashboard/home', {
      headers: {
        'Authorization': `Bearer ${localStorage.getItem('token')}`,
        'Content-Type': 'application/json'
      }
    });
    
    if (response.data.code === 10000) {
      const data = response.data.data;
      
      // 更新统计数据
      this.stats = {
        totalPets: data.statistics.totalPets,
        pendingApplications: data.statistics.pendingApplications,
        monthlyAdoptions: data.statistics.monthlyAdoptions,
        pendingFollowups: data.statistics.pendingFollowups
      };
      
      // 更新待办事项
      this.todos = data.todos;
      
      // 更新最近宠物
      this.recentPets = data.recentPets;
      
      // 更新最近申请
      this.recentApplications = data.recentApplications;
      
      // 更新回访提醒
      this.followupReminders = data.followupReminders;
      
      // 更新机构信息
      this.orgInfo = data.orgInfo;
      
      console.log('首页数据加载成功');
      return data;
    }
  } catch (error) {
    console.error('加载首页数据失败:', error);
    throw error;
  }
}
```

```javascript
        // 更新其他数据
        todos.value = data.todos;
        recentPets.value = data.recentPets;
        recentApplications.value = data.recentApplications;
        followupReminders.value = data.followupReminders;
        orgInfo.value = data.orgInfo;
        
        return data;
      } else {
        throw new Error(response.data.message);
      }
    } catch (err) {
      console.error('加载首页数据失败:', err);
      error.value = err.message;
      throw err;
    } finally {
      loading.value = false;
    }
  };
  
  // 格式化时间
  const formatTime = (time) => {
    if (!time) return '-';
    return new Date(time).toLocaleString('zh-CN');
  };
  
  // 处理待办事项点击
  const handleTodoClick = (todo) => {
    switch (todo.type) {
      case 'application':
        // 跳转到申请详情页
        router.push(`/org/applications/${todo.id}`);
        break;
      case 'followup':
        // 跳转到回访管理页
        router.push('/org/followup');
        break;
      case 'audit':
        // 跳转到宠物详情页
        router.push(`/org/pets/${todo.petId}`);
        break;
    }
  };
  
  return {
    // 数据
    stats,
    todos,
    recentPets,
    recentApplications,
    followupReminders,
    orgInfo,
    loading,
    error,
    
    // 方法
    loadDashboardData,
    formatTime,
    handleTodoClick
  };
}

// 在组件中使用
export default {
  setup() {
    const {
      stats,
      todos,
      loading,
      loadDashboardData,
      formatTime,
      handleTodoClick
    } = useDashboard();
    
    onMounted(() => {
      loadDashboardData();
    });
    
    return {
      stats,
      todos,
      loading,
      loadDashboardData,
      formatTime,
      handleTodoClick
    };
  }
};
```

## 10. 前端开发注意事项

### 10.1 数据加载策略
- **推荐使用综合接口** `/api/org/dashboard/home` 减少请求次数
- **添加加载状态**：避免用户重复点击

### 10.2 待办事项处理
- **优先级显示**：urgent > high > medium > low
- **时间排序**：按提交时间倒序排列
- **类型区分**：不同类型用不同图标和颜色区分
- **点击跳转**：根据待办类型跳转到对应页面

### 10.3 统计数据更新
- **定时刷新**：建议每隔5分钟自动刷新统计数据
- **手动刷新**：提供下拉刷新或刷新按钮
- **数据校验**：确保统计数据的一致性
- **本地缓存**：避免频繁请求相同数据

### 10.4 性能优化
- **图片懒加载**：宠物封面图片使用懒加载
- **分页加载**：列表数据建议分页加载
- **请求合并**：优先使用综合接口
- **CDN加速**：静态资源使用CDN

### 10.5 用户体验
- **加载动画**：显示加载动画提升用户体验
- **空状态处理**：无数据时显示友好的空状态提示
- **错误提示**：网络错误时提供重试按钮
- **数据更新**：操作后及时更新页面数据

### 11.2 统计数据SQL逻辑示例

```sql
-- 在养宠物总数
SELECT COUNT(*) FROM pet
WHERE org_user_id = ? AND status IN ('PUBLISHED', 'APPLYING') AND deleted = 0;

-- 待处理申请数
SELECT COUNT(*) FROM adoption_application aa
INNER JOIN pet p ON aa.pet_id = p.id
WHERE p.org_user_id = ? AND aa.status IN ('SUBMITTED', 'UNDER_REVIEW') AND aa.deleted = 0;

-- 本月领养数
SELECT COUNT(*) FROM adoption_application aa
INNER JOIN pet p ON aa.pet_id = p.id
WHERE p.org_user_id = ?
  AND aa.status = 'APPROVED'
  AND YEAR(aa.decided_time) = YEAR(CURRENT_DATE)
  AND MONTH(aa.decided_time) = MONTH(CURRENT_DATE)
  AND aa.deleted = 0;

-- 待回访数（已领养且超过7天未回访的宠物）
SELECT COUNT(DISTINCT aa.pet_id) FROM adoption_application aa
INNER JOIN pet p ON aa.pet_id = p.id
WHERE p.org_user_id = ?
  AND aa.status = 'APPROVED'
  AND aa.decided_time < DATE_SUB(CURRENT_DATE, INTERVAL 7 DAY)
  AND aa.deleted = 0
  AND NOT EXISTS (
    SELECT 1 FROM checkin_post cp
    WHERE cp.pet_id = aa.pet_id
      AND cp.create_time > DATE_SUB(CURRENT_DATE, INTERVAL 7 DAY)
      AND cp.deleted = 0
  );
```
