## 1. 页面功能概述

### 1.1 页面定位
创建宠物页面是机构端的核心功能，用于机构管理员创建和管理待领养宠物的档案信息。这是一个多步骤表单页面，支持宠物基本信息、健康状况、性格特征、领养要求等完整信息的录入。

### 1.2 核心功能模块

| 模块 | 功能描述 | 权限要求 |
|------|---------|---------|
| **基本信息** | 宠物名称、物种、品种、年龄、性别、体型、毛色等 | `pet:create` |
| **健康状况** | 疫苗、绝育、驱虫状态及健康描述 | `pet:create` |
| **性格特征** | 性格描述、特征标签选择 | `pet:create` |
| **领养要求** | 领养条件和要求说明 | `pet:create` |
| **图片/视频上传** | 上传宠物照片（3-9张）和视频 | `pet:create` |
| **地理位置** | 显示/编辑宠物所在位置（默认继承机构地址） | `pet:create` |
| **保存草稿** | 保存为草稿状态，可继续编辑 | `pet:create` |
| **提交审核** | 提交到管理员审核 | `pet:submit_audit` |
| **预览** | 预览宠物信息展示效果 | `pet:create` |

## 3. 接口需求分析

### 3.1 必需接口（6个）

#### 3.1.1 创建宠物档案
- **路径**: `POST /api/org/pets`
- **权限**: `pet:create`
- **优先级**: 🔴 高（核心接口）

#### 3.1.2 公共文件上传 ⭐
- **路径**: `POST /api/upload`
- **权限**: 需要认证
- **优先级**: 🔴 高（核心功能）
- **说明**: 使用系统统一的文件上传接口，支持多种文件类型

#### 3.1.3 保存宠物媒体关联
- **路径**: `POST /api/org/pets/:id/media`
- **权限**: `pet:create`
- **优先级**: 🔴 高（核心功能）
- **说明**: 将已上传的文件URL与宠物关联

#### 3.1.4 提交审核
- **路径**: `POST /api/org/pets/:id/submit-audit`
- **权限**: `pet:submit_audit`
- **优先级**: 🔴 高（核心业务）

#### 3.1.5 获取机构信息（用于默认地址）
- **路径**: `GET /api/org/profile`
- **权限**: `org:profile:read`
- **优先级**: 🟡 中（优化体验）

#### 3.1.6 获取标签列表（用于特征选择）
- **路径**: `GET /api/tags`
- **权限**: 无需认证
- **优先级**: 🟡 中（优化体验）

### 3.2 可选接口（2个）

#### 3.2.1 更新宠物信息（编辑模式）
- **路径**: `PUT /api/org/pets/:id`
- **权限**: `pet:update`
- **优先级**: 🟡 中（编辑功能）

#### 3.2.2 删除宠物媒体
- **路径**: `DELETE /api/org/pets/:id/media/:mediaId`
- **权限**: `pet:update`
- **优先级**: 🟢 低（优化体验）

### 4.2 计算属性

```typescript
// 计算年龄（月）
const ageMonth = computed(() => {
  return formData.value.ageYears * 12 + formData.value.ageMonths
})

// 是否有封面图
const hasCoverImage = computed(() => {
  return mediaList.value.some(m => m.isCover && m.mediaType === 'IMAGE')
})

// 图片数量
const imageCount = computed(() => {
  return mediaList.value.filter(m => m.mediaType === 'IMAGE').length
})

// 是否可以提交
const canSubmit = computed(() => {
  // 必须有封面图
  if (!hasCoverImage.value) return false
  // 图片数量3-9张
  if (imageCount.value < 3 || imageCount.value > 9) return false
  // 基本信息必填
  if (!formData.value.name || !formData.value.breed) return false
  return true
})

// 是否显示提交审核按钮
const showSubmitButton = computed(() => {
  return currentStep.value === 4 && canSubmit.value
})

// 表单标题
const formTitle = computed(() => {
  return isEditMode.value ? '编辑宠物' : '创建宠物'
})
```

### 4.3 核心方法

```typescript
// 加载机构信息
const loadOrgInfo = async () => {
  try {
    const response = await orgAPI.getOrgProfile()
    orgInfo.value = response.data

    // 设置默认地理位置
    if (orgInfo.value.lng && orgInfo.value.lat) {
      formData.value.location = {
        address: orgInfo.value.address || '',
        city: orgInfo.value.city || '',
        district: orgInfo.value.district || '',
        lng: orgInfo.value.lng,
        lat: orgInfo.value.lat
      }
    }
  } catch (error) {
    console.error('加载机构信息失败:', error)
  }
}

// 加载标签列表
const loadTagList = async () => {
  try {
    const response = await orgAPI.getTagList()
    // 按类型分组
    response.data.list.forEach(tag => {
      if (tagList.value[tag.tagType]) {
        tagList.value[tag.tagType].push(tag)
      }
    })
  } catch (error) {
    console.error('加载标签列表失败:', error)
  }
}

// 上传图片
const uploadImage = async (file) => {
  if (imageCount.value >= 9) {
    ElMessage.warning('最多只能上传9张图片')
    return
  }

  try {
    // 步骤1: 使用公共上传接口上传文件
    const uploadFormData = new FormData()
    uploadFormData.append('file', file)
    uploadFormData.append('type', 'pet_photo')

    const uploadResponse = await uploadAPI.uploadFile(uploadFormData, {
      onUploadProgress: (progressEvent) => {
        const percentCompleted = Math.round(
          (progressEvent.loaded * 100) / progressEvent.total
        )
        // 更新上传进度
      }
    })

    // 步骤2: 如果已有宠物ID，保存媒体关联信息
    let mediaId = null
    if (petId.value) {
      const mediaResponse = await orgAPI.savePetMedia(petId.value, {
        url: uploadResponse.data.url,
        mediaType: 'IMAGE',
        sort: mediaList.value.length
      })
      mediaId = mediaResponse.data.id
    }

    // 添加到媒体列表
    mediaList.value.push({
      id: mediaId,
      url: uploadResponse.data.url,
      mediaType: 'IMAGE',
      sort: mediaList.value.length,
      isCover: !hasCoverImage.value,  // 第一张图片自动设为封面
      isUploading: false,
      uploadProgress: 100
    })

    ElMessage.success('图片上传成功')
  } catch (error) {
    console.error('图片上传失败:', error)
    ElMessage.error('图片上传失败，请重试')
  }
}

// 删除媒体
const removeMedia = async (index) => {
  const media = mediaList.value[index]

  // 如果是已上传的媒体，调用删除接口
  if (media.id && petId.value) {
    try {
      await orgAPI.deletePetMedia(petId.value, media.id)
    } catch (error) {
      console.error('删除媒体失败:', error)
      return
    }
  }

  // 从列表中移除
  mediaList.value.splice(index, 1)

  // 如果删除的是封面图，设置第一张为封面
  if (media.isCover && mediaList.value.length > 0) {
    mediaList.value[0].isCover = true
    coverIndex.value = 0
  }
}

// 设置封面图
const setCoverImage = (index) => {
  mediaList.value.forEach((media, i) => {
    media.isCover = i === index
  })
  coverIndex.value = index
}

// 保存草稿
const saveDraft = async () => {
  try {
    submitting.value = true

    const data = {
      ...formData.value,
      ageMonth: ageMonth.value,
      status: 'DRAFT',
      auditStatus: 'NONE',
      lng: formData.value.location.lng,
      lat: formData.value.location.lat,
      coverUrl: mediaList.value.find(m => m.isCover)?.url || ''
    }

    let response
    if (isEditMode.value && petId.value) {
      // 更新
      response = await orgAPI.updatePet(petId.value, data)
    } else {
      // 创建
      response = await orgAPI.createPet(data)
      petId.value = response.data.id
    }

    ElMessage.success('草稿保存成功')

    // 如果是首次保存，切换到编辑模式
    if (!isEditMode.value) {
      isEditMode.value = true
      // 上传已选择的图片
      await uploadMediaList()
    }
  } catch (error) {
    console.error('保存草稿失败:', error)
    ElMessage.error('保存失败，请重试')
  } finally {
    submitting.value = false
  }
}

// 提交审核
const submitForAudit = async () => {
  // 先保存草稿
  if (!petId.value) {
    await saveDraft()
  }

  // 验证必填项
  if (!canSubmit.value) {
    ElMessage.warning('请完善所有必填信息')
    return
  }

  try {
    submitting.value = true

    await orgAPI.submitPetAudit(petId.value)

    ElMessage.success('提交审核成功')
    router.push('/org/pets')
  } catch (error) {
    console.error('提交审核失败:', error)
    ElMessage.error('提交失败，请重试')
  } finally {
    submitting.value = false
  }
}

// 上传媒体列表
const uploadMediaList = async () => {
  if (!petId.value) return

  for (let i = 0; i < mediaList.value.length; i++) {
    const media = mediaList.value[i]

    if (media.isUploading || media.id) continue  // 跳过已上传或正在上传的

    media.isUploading = true

    try {
      const formData = new FormData()
      formData.append('file', media.file)
      formData.append('type', media.mediaType)
      formData.append('sort', i)

      const response = await orgAPI.uploadPetMedia(petId.value, formData)

      media.url = response.data.url
      media.id = response.data.id
      media.isUploading = false
      media.uploadProgress = 100
    } catch (error) {
      console.error('上传媒体失败:', error)
      media.isUploading = false
    }
  }
}

// 切换标签
const toggleTag = (tag) => {
  const index = formData.value.personality.tags.indexOf(tag.id)
  if (index > -1) {
    formData.value.personality.tags.splice(index, 1)
  } else {
    formData.value.personality.tags.push(tag.id)
  }
}

// 上一步
const prevStep = () => {
  if (currentStep.value > 1) {
    currentStep.value--
  }
}

// 下一步
const nextStep = async () => {
  // 验证当前步骤
  if (!validateStep(currentStep.value)) {
    return
  }

  if (currentStep.value < 4) {
    currentStep.value++
  }
}

// 验证步骤
const validateStep = (step) => {
  if (step === 1) {
    // 验证基本信息
    if (!formData.value.name || !formData.value.breed) {
      ElMessage.warning('请填写宠物名称和品种')
      return false
    }
  }
  return true
}

// 预览
const handlePreview = () => {
  // 保存草稿
  saveDraft()

  // 跳转到预览页面
  router.push(`/org/pets/${petId.value}/preview`)
}
```

### 7.5 性能优化

#### 图片压缩
```javascript
const compressImage = async (file) => {
  return new Promise((resolve, reject) => {
    const reader = new FileReader()
    reader.onload = (e) => {
      const img = new Image()
      img.onload = () => {
        const canvas = document.createElement('canvas')
        const ctx = canvas.getContext('2d')

        // 计算压缩后的尺寸
        let width = img.width
        let height = img.height
        const maxSize = 1920

        if (width > maxSize || height > maxSize) {
          const ratio = Math.min(maxSize / width, maxSize / height)
          width *= ratio
          height *= ratio
        }

        canvas.width = width
        canvas.height = height
        ctx.drawImage(img, 0, 0, width, height)

        // 压缩并导出
        canvas.toBlob(
          (blob) => {
            resolve(new File([blob], file.name, { type: 'image/jpeg' }))
          },
          'image/jpeg',
          0.8
        )
      }
      img.onerror = reject
      img.src = e.target.result
    }
    reader.onerror = reject
    reader.readAsDataURL(file)
  })
}
```
