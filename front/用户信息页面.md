# 用户信息页面 - 前端接口对接文档

> 本文档提供用户信息页面所需的所有后端接口，包括请求参数、响应格式、字段说明等，方便前端开发对接。

## 1. 获取用户详细信息

**接口说明**：获取当前登录用户的详细信息，包含信用分、统计数据、徽章等信息

**请求方式**：`GET`

**请求路径**：`/api/users/profile`

**请求头**：
```
Authorization: Bearer <access_token>
Content-Type: application/json
```

**请求参数**：无（从token中获取用户ID）

**响应示例**：
```json
{
  "code": 10000,
  "message": "操作成功",
  "data": {
    "userId": 12345,
    "username": "zhangsan",
    "nickname": "zhangsan",
    "avatar": "https://example.com/avatar.jpg",
    "role": "USER",
    "phone": "13800138000",
    "email": "zhangsan@example.com",
    "city": null,
    "creditScore": 85,
    "creditChange": 12,
    "creditLevel": "银牌",
    "medals": ["guardian", "adopter", "verified"],
    "stats": {
      "applications": 5,
      "favorites": 12,
      "checkins": 30,
      "adoptions": 2,
      "pendingApplications": 1,
      "monthlyCheckins": 30
    },
    "createTime": "2024-01-01 10:00:00"
  }
}
```

**响应字段说明**：

| 字段 | 类型 | 说明 |
|------|------|------|
| userId | Long | 用户ID |
| username | String | 用户名 |
| nickname | String | 昵称（当前与用户名相同） |
| avatar | String | 头像URL |
| role | String | 用户角色（USER/ORG/ADMIN） |
| phone | String | 手机号 |
| email | String | 邮箱 |
| city | String | 城市 |
| creditScore | Integer | 当前信用分 |
| creditChange | Integer | 信用分变化（正数为增加，负数为减少） |
| creditLevel | String | 信用等级（新注册/铜牌/银牌/金牌/铂金） |
| medals | Array | 徽章列表（guardian/adopter/verified） |
| stats | Object | 统计数据 |
| stats.applications | Integer | 申请总数 |
| stats.favorites | Integer | 收藏总数 |
| stats.checkins | Integer | 打卡总数 |
| stats.adoptions | Integer | 领养总数 |
| stats.pendingApplications | Integer | 待处理申请数 |
| stats.monthlyCheckins | Integer | 本月打卡数 |
| createTime | String | 注册时间 |

## 12. 前端对接示例

### 12.1 获取用户详细信息

```javascript
// 使用axios请求
import axios from 'axios';

async function getUserProfile() {
  try {
    const response = await axios.get('/api/users/profile', {
      headers: {
        'Authorization': `Bearer ${localStorage.getItem('token')}`
      }
    });
    
    if (response.data.code === 10000) {
      const userProfile = response.data.data;
      // 处理用户信息
      console.log('用户信息:', userProfile);
      return userProfile;
    } else {
      console.error('获取用户信息失败:', response.data.message);
    }
  } catch (error) {
    console.error('请求失败:', error);
  }
}

// 调用
getUserProfile();
```

### 12.2 更新用户偏好设置

```javascript
async function updateUserPreference(preference) {
  try {
    const response = await axios.put('/api/users/preference', preference, {
      headers: {
        'Authorization': `Bearer ${localStorage.getItem('token')}`,
        'Content-Type': 'application/json'
      }
    });
    
    if (response.data.code === 10000) {
      console.log('偏好设置更新成功');
      return true;
    } else {
      console.error('更新失败:', response.data.message);
      return false;
    }
  } catch (error) {
    console.error('请求失败:', error);
    return false;
  }
}

// 调用
updateUserPreference({
  petTypes: ['CAT', 'DOG'],
  ageRange: [0, 36],
  gender: 'MALE',
  tags: ['亲人', '活泼'],
  distance: 50,
  sizes: ['S', 'M'],
  healthRequirements: ['已绝育', '已疫苗']
});
```

### 12.3 获取信用流水（分页）

```javascript
async function getCreditLogs(pageNo = 1, pageSize = 10) {
  try {
    const response = await axios.get('/api/credit/logs', {
      params: { pageNo, pageSize },
      headers: {
        'Authorization': `Bearer ${localStorage.getItem('token')}`
      }
    });
    
    if (response.data.code === 10000) {
      const { list, total, totalPages } = response.data.data;
      console.log(`信用流水: 共${total}条，${totalPages}页`);
      return response.data.data;
    }
  } catch (error) {
    console.error('请求失败:', error);
  }
}

// 调用
getCreditLogs(1, 10);
```

## 13. 注意事项

1. **认证**：所有接口都需要在请求头中携带 `Authorization: Bearer <token>`
2. **时间格式**：所有时间字段使用 ISO 8601 格式（如 `2024-02-14T10:30:00`）
3. **分页**：分页接口的 `pageNo` 从 1 开始
4. **错误处理**：所有接口返回统一的 `code` 字段，`10000` 表示成功，其他表示失败
5. **数据脱敏**：手机号、邮箱等敏感信息已脱敏处理
6. **权限控制**：部分接口需要特定权限，请确保用户已登录并拥有相应权限
